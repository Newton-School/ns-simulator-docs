# Provider Mapping Notes

## Mapping dimensions
- Compute runtime
- Container orchestration
- Relational and NoSQL storage
- Object and block storage
- Messaging and streaming
- Secrets and identity
- Observability stack

## Cost analysis checklist
- Baseline steady-state cost
- Spike and burst behavior cost
- Cross-region data transfer cost
- Operational overhead cost assumptions

## Tradeoff checklist
- Latency sensitivity
- Durability requirements
- Compliance constraints
- Portability and lock-in risk
